서울 교통·내비 완전 가이드 — 지하철·버스·택시·앱 모두 해결

> 한국 교통은 어렵지 않습니다. 이 가이드 하나로 서울이 당신 손에 들어옵니다.

한눈에 보기

항목 정보
대중교통 지하철·버스·택시 모두 저렴 (1회 기본요금 1,250~2,500원)
교통카드 T-money 또는 Climate Card (편의점에서 구매 가능)
주요 앱 네이버지도, 카카오맵, 서울교통공사 공식앱
영어/중국어 주요 역·앱 모두 지원
24시간 지하철은 자정~5시 운행 중단, 심야버스·택시는 24시간
비용 편리함 vs 가성비 선택 (택시 = 비쌈, 지하철 = 저렴)

Part 1: 교통카드 — T-money vs Climate Card

꼭 필요한가? 필수입니다. 현금으로 매번 사면 손해.

T-money (티머니)

항목 설명
구매처 GS25, CU, 이마트, 편의점 모두 가능
구매가 카드값 2,500원 + 잔액 충전 (최소 2,000원)
사용처 지하철·버스·택시·편의점 모두 가능
충전 편의점·역 자동충전기 (24시간)
환불 출국 전 환전 가능 (손수료 500원)

장점: 한국 전역 호환. 최고의 선택.

Climate Card (클라이밋 카드)

항목 설명
구매처 GS25, CU 등 편의점 (T-money와 동일)
구매가 카드값 2,500원 + 잔액 (최소 2,000원)
사용처 지하철·버스·택시·편의점 (T-money와 동일)
특징 환경 이슈(재활용 가능)로 도입됨

T-money와의 차이: 거의 없음. 편의점에 있는 것 아무거나 구매해도 됨.

교통카드 사용 팁

1. 구매할 때: "T-money 카드 주세요"라고 하거나 손가락으로 지적

2. 충전: 편의점 카운터 또는 역의 자동충전기 (자동충전기가 쉬움)

3. 사용: 지하철·버스 탈 때 카드리더기에 접촉 (삐 소리)

4. 잔액 확인: 편의점 카운터 또는 앱

5. 환불: 공항 또는 출국 전 환전소에서 (카드값 - 수수료 반환)

Part 2: 지하철 완벽 가이드

서울 지하철 기본 정보

항목 설명
노선 수 9개 주요 노선 (1~9호선) + 분기선
운행 시간 기본 05:30~자정 (노선마다 다름)
심야 운행 없음 (자정~05:30 운행 중단, 심야버스 이용)
기본요금 1,250원 (10km 이내)
추가요금 10km 초과 시 100원씩 추가
혼잡도 평일 오전 8~9시, 오후 5~6시 최고

지하철 역 내 시설

시설 설명
에스컬레이터 왼쪽 탈 것 (오른쪽은 비워두는 문화)
화장실 모든 역에 있음 (일부 비밀번호, 보통 무료)
편의점 GS25·CU (모든 역 지하상가)
카페 역 주변 또는 지하상가
안내소 각 역마다 있음 (직원 영어 가능)

지하철 탑승 완벽 프로세스

Step 1: 목적지 찾기

방법 1: 스마트폰 앱

  • 네이버지도 또는 카카오맵 열기
  • "출발지 → 목적지" 입력
  • "대중교통" 탭 선택
  • 추천 경로 따라가기

방법 2: 역 안내도

  • 역사 내 큰 지하철 노선도 찾기
  • 현재 역 확인 (보통 빨간 점)
  • 목적지 확인
  • 필요 노선 찾기

Step 2: 역 입장

1. 개표구 또는 자동 개표기 찾기

2. T-money 카드 또는 일회용 카드 구매

3. 자동 개표기에 카드 접촉 (삐 소리 = 정상)

4. 게이트 통과

Step 3: 플랫폼 이동

1. 승강장 방향 표지판 따라가기 (보통 위쪽 화살표)

2. "1번선" 또는 "2번 방향" 확인

3. 해당 방향 플랫폼 이동

Step 4: 탑승

1. 플랫폼에서 기다리기

2. 열차 도착 (디지털 시계판에 남은 시간 표시)

3. 문이 열리면 탑승 (내릴 사람 먼저 내려주기)

4. 손잡이 잡기 (흔들림 많음)

Step 5: 내리기

1. 역 이름 확인 (차 내 디지털 표지판 + 음성 안내)

2. 원하는 역 도착 시 준비

3. 문이 열리면 내리기

4. "출구 1, 2, 3" 확인해서 나가기

Step 6: 역 탈출

1. 안내도 또는 스마트폰 맵으로 출구 확인

2. 에스컬레이터 또는 계단 이용

3. 개표구 통과 (카드 접촉)

지하철 환승

같은 역에서 환승하는 경우:

  • 보통 가능 (표지판 따라가기)
  • 추가 요금 없음 (1시간 내 환승)

다른 역에서 환승하는 경우:

  • 불가능 (같은 역 내에서만 환승 가능)

환승 제한 시간:

  • 일반: 1시간 이내 (추가 요금 없음)
  • 초과: 추가 요금 발생

지하철 편리한 팁

팁 설명
헷갈리는 노선도 스마트폰 맵 앱만 믿기 (더 정확함)
방향 헷갈림 "Incheon" 또는 "Myeongdong" 방향 찾기
장시간 운행 1호선은 40분+, 2호선은 60분+ (매우 김)
짐이 많을 때 맨 끝 객차가 짐 공간 많음
여성 전용 칸 평일 저녁 8~10시 지정 (탑승선 구분됨)
야경 명소 2호선 타고 강남~을지로 구간 추천

Part 3: 버스 가이드

버스 종류

1. 일반 버스 (파란색)

항목 설명
요금 1,250원 (지하철과 동일)
속도 느림 (정류장 많음)
추천 짧은 거리 + 경치 좋은 코스

탈 때:

  • 앞문에서 탑승 + 카드 접촉
  • 뒷문에서 하차

2. 좌석버스 (빨강/초록색, 중앙버스전용차로)

항목 설명
요금 1,450~2,500원 (거리별)
속도 빠름 (중앙버스전용차로 이용)
추천 먼 거리 + 시간 부족할 때

특징:

  • 중앙의 전용 차로에서만 운행 (빠름)
  • 좌석이 있어 쾌적함 (일반버스는 서야 할 수 있음)
  • 거리에 따라 요금 다름

버스 탑승 프로세스

1. 버스 정류소에서 기다리기: 번호판 확인

2. 버스 도착: 손 들고 기사에게 신호 (안 하면 안 멈출 수 있음)

3. 탑승: 앞문으로 탑승

4. 카드 접촉: 리더기에 T-money 카드 접촉 (삐 소리)

5. 자리 앉기 또는 손잡이 잡기

6. 목적지 확인: 차량 앞 디지털 표지판에서 역명 확인

7. 내리기: 표지판에 목적지 역 떠오르면 "내립니다" 손 들기

8. 뒷문으로 하차: 카드 재접촉 (일부 버스)

버스 앱 확인법

네이버지도 또는 카카오맵:

1. 버스 정류소 입력

2. "버스" 탭 선택

3. 실시간 버스 위치 확인

4. 몇 정류소 남았는지 확인

5. 예상 도착시간 확인

Part 4: 택시 — 편리하지만 비쌈

택시 기본 정보

항목 설명
기본요금 3,400원 (기본 2km)
추가요금 100m마다 100원 추가
시간요금 35초마다 100원 (정체 시)
심야요금 자정~4시 +20% 할증
일반요금 매우 비쌈 (5km 거리에 15,000원+)

택시 종류

종류 색상 가격 특징
일반 택시 은색/노란색 기본 가장 흔함
모범 택시 검정색 +20% 비쌈 깔끔함, 영어 가능한 기사
대형 택시 회색 +50% 비쌈 승객 4명 이상
택시앱 모두 변동 카카오T, 우버 (가격 변동)

택시 탑승 방법

길거리에서 잡기

1. 손 들어 신호: 기사가 보도록 손을 올리기

2. 택시 정차: 기사가 보면 정지선에 정차

3. 목적지 말하기: 영어 또는 주소 말하기 (못 알아들으면 핸드폰에서 주소 보여주기)

4. 탑승: 뒷좌석에 탑승

5. 요금 확인: 미터기 켜져 있는지 확인 (중요!)

6. 목적지 도착: 신용카드로 결제 (또는 현금)

7. 팁: 한국은 팁 문화 없음 (거스름돈도 받기)

택시앱으로 부르기

카카오T (가장 인기):

```

1. 앱 설치 및 로그인

2. "출발지" 입력 (자동으로 현재 위치)

3. "목적지" 입력

4. "택시 예약" 클릭

5. 택시 도착 대기

6. 핸드폰에 기사 정보 표시

7. 탑승

8. 앱에서 결제 (카드 연동)

```

장점:

  • 미리 가격 예측 가능
  • 신용카드 자동 결제 (현금 필요 없음)
  • 택시 위치 실시간 추적
  • 기사 정보 확인 가능

단점:

  • 거리 요금보다 비쌀 수 있음 (변동요금제)
  • 앱 설치 필요

택시 팁 & 주의사항

항목 설명
미터기 반드시 켜져 있는지 확인. 안 켜져 있으면 탑승하지 말 것
거스름돈 기사에게 물어봤을 때 챙기기 (한국에서 팁 안 줘도 됨)
무승차거부 불법 (가까운 거리여도 탑승해야 함)
카드 결제 신용카드 가능 (기사가 POS기 꺼내옴)
외국인 영어 능력 기사가 적음. 주소를 핸드폰에 쓰는 것이 최선
야경 드라이브 간선도로(한강 주변) 택시 타고 야경 감상 좋음

Part 5: 네비게이션 앱 — 완벽 가이드

네이버 지도 vs 카카오맵

네이버 지도

장점:

  • 실시간 교통 정보 정확함
  • 대중교통 길찾기 가장 정확
  • 영어 지원 완벽

단점:

  • 중국에서는 접근 불가
  • 일부 소규모 가게 정보 부족

추천: 대부분의 여행객. 특히 지하철/버스 이용 시.

카카오맵

장점:

  • 중국에서 접근 가능 (VPN 필요 없음)
  • 음식점·카페 정보 풍부
  • 실시간 택시 호출 가능 (카카오T 연동)

단점:

  • 영어 지원 상대적으로 약함
  • 대중교통 경로 때때로 부정확

추천: 중국 관광객, 음식점 정보 많이 필요할 때

네이버 지도 사용 팁

```

1. 앱 설치 및 한국어 설정

2. 우측 상단 "언어" → "English" 변경

3. 검색창에 "지역명" 또는 "주소" 입력

4. "길찾기" 선택

5. 교통 수단 선택 (지하철 / 버스 / 택시 / 자동차 / 자전거)

6. "출발" 버튼

7. 실시간 길찾기 시작

```

실시간 교통 정보 읽기

네이버 지도 화면:

표시 의미
녹색 원활한 교통
노란색 약간 막힘
빨간색 심한 정체
회색 운행 정보 없음

중국 관광객 특화: 위챗맵 (WeChat Map)

뭔가? 중국의 바이두 지도·높은더 지도와 유사. 위챗 내에 내장됨.

장점:

  • 중국에서 개발 (호환성 좋음)
  • 위챗페이 연동

단점:

  • 한국 데이터 부족 (추천하지 않음)

결론: 카카오맵이 차선책.

Part 6: 따릉이 (따릉이) — 공용자전거

따릉이란?

뭔가? 서울의 공용자전거 시스템. 전역에 대여소가 있음.

항목 설명
대여료 30분 이내 1,000원 (추가 10분마다 500원)
가입 앱(따릉이앱) 또는 대여소 직접 가입
자격 신용카드 필요 (보증금 개념)
특징 서울 곳곳 1,400+대여소

따릉이 이용법

1. 앱 설치: "따릉이" 앱 다운로드

2. 회원가입: 신용카드 등록 (신청 당일 불가, 1일 소요)

3. 대여소 찾기: 지도에서 대여소 확인

4. 대여: 앱에서 자전거 번호 입력 후 잠금 해제

5. 반납: 대여소의 빈 자리에 자전거 끼우기 (자동 잠금)

Part 7: 공항 이동 완벽 가이드

인천공항 → 서울 시내

교통수단 소요시간 가격 추천도
공항철도 60분 9,000원 ⭐⭐⭐⭐
공항버스 70~120분 10,000~15,000원 ⭐⭐⭐⭐
리무진 60~90분 25,000~35,000원 ⭐⭐⭐
택시 60~90분 70,000~100,000원 ⭐

추천 조합:

  • 혼자 또는 2명: 공항철도 (가성비 + 빠름)
  • 3명 이상: 리무진 (짐 많을 때)
  • 특정 호텔 묵을 때: 공항버스 (호텔 직행)

공항철도 이용법

```

1. 인천공항 도착 → 기차역 표지판 따라가기

2. 공항철도 표 구매 (자동판매기 또는 카운터, 9,000원)

3. 게이트 통과 (표 제시)

4. 기차 탑승 (서울역 또는 인천공항철도 노선 역행)

5. 서울역 도착 (소요 60분)

6. 지하철 환승 (서울역은 1·4호선 연결)

```

외국인 여행객 체크리스트

```

[ ] T-money 카드 구매 (편의점)

[ ] 충전 (최소 5,000원)

[ ] 네이버 지도 또는 카카오맵 설치

[ ] 스마트폰 한글 입력 설정 (주소 검색용)
```
